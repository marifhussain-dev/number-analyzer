# Number Analyzer (Mini Project)

A Biginner-friendly python mini project build to practice python fundamentals
## Description
A Python CLI application that:
- Authenticates users with a login system(3 attempts)
- Collect numbers until the user enters 0
- Calculate total, count, average, minimum, maximum

## Features
- Login authentication
- Input validation
- Modular Functions
- Handles edge cases

## Concept Used
- Functions
- while loops
- for loops
- Lists
- Conditionals
