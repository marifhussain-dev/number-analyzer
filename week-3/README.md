# Week 3 — Day 1: File Stats Mini Tool

This script reads numbers from `data.txt`, ignores invalid lines, computes stats, and saves results to `summary.txt`.

## Files
- `week3_day1_final.py` — main script
- `data.txt` — sample input
- `summary.txt` — output (generated by script)

## How to run
```bash
python week3_day1_final.py

## What it does
- Skips invalid lines(example: abc)
- Print: Count, Sum, Min, Max, Avg
- Write the same stats to summary.txt
